# ENGR103 Solar Panel Project

This Python project calculates daily solar energy output in kilowatt-hours (kWh).

## Features
- Temperature derate factor
- Effective power per panel
- Total array power
- Daily energy output

## Example Results
Worked Example: 15.5 kWh
Test Case 1: 14.6 kWh
Test Case 2: 9.2 kWh
